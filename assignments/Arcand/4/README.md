Assignment Description:

I created a README.md report containing a comprehensive table summarizing the framability practices of the 100 unique, random websites from Mozilla's Top 500 Sites I was assigned in addition to demonstrating how that Path attribute for Cookies is not suitable for security.

The files are saved in the Arcand\4 folder.

Websites.txt is a list of the 100 unique, random websites from Mozilla's Top 500 Sites I was assigned. README.md helps explain the assignment and provide documentation.
framable_results.csv is the result of running checkFramability.js on all of the framable files in the framable folder created by running generateHTML.js with Websites.txt.
server.js starts the server.
generateHTML.js was used to generate the HTML files in the framable folder.
checkFramability.js was used to check the framability of the websites. I also checked the results manually.
